# BuildPro - Professional Construction Services

A responsive landing page for a construction services company.

## Features

- Responsive design that works on desktop and mobile
- Hero section with email quote request form
- Services showcase (Residential, Commercial, Renovations)
- Clean, modern UI with green color scheme

## Getting Started

Open `index.html` in your browser to view the site.

## Project Structure

```
├── index.html    # Main landing page
├── styles.css    # Stylesheet
└── README.md
```

## Technologies

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
